Zipcode

Data Object Parameters

  • @ZipID
  • @Zipcode

Description

The Zipcode table is used to hold the ZipID and Zipcode of a donor (FoodSource) or Agency.

The ZipID is an auto-incrementing smallint, which the database creates upon creation of an address a state.

The Zipcode holds a zipcode's digits. The database can hold up to a varchar of length 10.
