Masterpage Inheritance

Masterpages provide an "inheritance template" for pages that inherit from it. They are used in this project to provide graphical unity and user authentication across the application.

Masterpages

global.master

The masterpage global.master provides a "user authentiaction umbrella" for the entire project, as well as session time-out managment. It does not provide any HTML structure, but does include the background CCS watermark image.

Mobile.master

All tablet pages inherit from Mobile.master. It provides the flexible width container, top menu, and other styles inherited by all tablet pages. Additionally, many desktop pages inherit from mobile.master styles.

Desktop.master

Desktop designed pages (reports, user admin, etc) inherit from Desktop.master, which provides a style more appropriate for desktop width functions.
