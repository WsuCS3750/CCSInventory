Audit

The audit functionality, located in the audit folder, provides the ability quickly as easily inspect each container in the warehouse and either verify or correct their contents.

Audit Pages

default.aspx

This page is used to show the user all audits that have been performed. They can click the accompanying "adjustments" button to view any corrections that were made during the audit. Using the "Perform Audit" button launches an audit session.

view.aspx

This page displays the adjustments made during the audit process.

perform.aspx

This page uses two GridViews. The first displays all containers that haven't yet been either verified or corrected. The second gridview shows all adjustments that were made to containers during the audit. The user can either select the "Verify" button next to a container or enter the container's Container Number to be brought the the verify.aspx page.

verify.aspx

This page displays the information about the container. If the information is changed, then the "Verify" button turns into a "Save" button, meaning that the container's contents are incorrect and need to be adjusted.
