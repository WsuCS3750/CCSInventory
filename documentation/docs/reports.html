Report

There are five types of Reports:

In\Out

This shows how much of a category had incoming and outgoing. Also it will show the current inventory on hand for the category.

Incoming

This report shows all the food in records between two specified dates.

Inventory

This shows all the current food in containers in the database ordery by location or category.

Grocery Rescue

This is a report that CCS has to turn in weekly that states all the food they acquired through businesses that are partcipating in the Grocery Rescue program. This food is ussually perishable.

Outgoing

This report shows all the food out records between two specified dates.

Report Creation

choosetemplate.aspx

The report type is passed in the query string to this page. This specifies how the pages will flow for the report creation. This page shows the user previous templates they made of the specified report they chose. They can run, edit, or delete any of these templates. If they choose edit they will be able edit all the options in the report. They will also be able to create a whole new report that will ask them all the options for the specified report. At the end theses options they will be asked to save the report as a template.

Options

There are various options throughout each report type. Most the reports allows the user to select certian categories to show on the report such as agencies, food categories, etc. Also, at the end of each options there is a specific option for the report type that allows the user to specify a view, a name, and possibly a few other options depending on the eport.

save.aspx

The page will allow the user to save the report they just specified as a template, so that it may be use later quickly. They can specifiy a name then in the code it will serialize all the report options specified by using the template class (stored in AppCode) and save it to the database.

Displays

Each Report has a display page that queries the data specified the template then generates the report via Windows Reports.
