Donor Service

Description

Provides a JavaScript array of all food sources suitable for use in an autocomplete or other JavaScript function.

Array Sample

donorNames = [{label: 'Clark Kent', address: '125 Awesome Road', type: 'Individual' },
    {label: 'Elmer Fudd', address: 'jlkfdjdlsf', type: 'Individual' }];

Usage

<script src="../js/jquery-ui-1.10.0.custom.min.js" type="text/javascript"></script>
<link href="../css/ui-lightness/jquery-ui-1.10.0.custom.min.css" rel="stylesheet" />

<script src="DonorService.ashx" type="text/javascript"></script>
    
<script>
    $(document).ready(function () {
        $('#<%= txtDonorName.ClientID %>').autocomplete({ source: donorNames });
    });
</script>
