State

Data Object Parameters

  • @StateID
  • @StateFullName
  • @StateShortName

Description

The State table is used to hold the StateID, StateFullName, and StateShortName of a donor (FoodSource) or Agency.

The StateID is an auto-incrementing smallint, which the database creates upon creation of an address a state.

The StateFullName holds a state's complete name; i.e. Utah. The database can hold up to a varchar of length 25.

The StateShortName holds a state's 2 character abbreviation; i.e. UT. The database can hold up to a varchar of length 2.
