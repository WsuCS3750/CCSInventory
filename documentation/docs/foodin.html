Food In

AKA Incoming Food

Data Object Parameters

  • @FoodInID
  • @TimeStamp
  • @Weight
  • @Units
  • @FoodSourceID
  • @CategoryID
  • @USDAID

Description

FoodIn table is used to log the food being received.

FoodInID field is an auto-incrementing number.

TimeStamp is used to log the time and date the donation was received.

Weight is used the total weight of the donation.

Units is used to track the number of cases received. Units is a nullable field.

FoodSourceID is used to link a foodsource (donor) to the donation being received.

CategoryID is used to link a category (food type) to a donation, in order to specify what the received donation is. The CategoryID is nullable.

USDAID is used to link a donation to a particular USDA category, in order to specify what the received donation is.
The USDAID is nullable and is only used for donations from the USDA donor.
