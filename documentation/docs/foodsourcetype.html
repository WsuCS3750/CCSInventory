Food Source Type

AKA Donor Type

Data Object Parameters

  • @FoodSourceTypeID
  • @FoodSourceType

Description

FoodSourceType table is used to hold the donor type For example: Grocery Rescue or In-Kind Donation.

FoodSourceTypeID links a donor to a donor type. It is an auto-incrementing number

FoodSourceType specifies the name of the food source type; i.e. In-Kind or Grocery Rescue.
The field takes a length up to 50 of type varchar.
