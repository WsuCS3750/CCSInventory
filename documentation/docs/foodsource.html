Food Source

AKA Donor

Data Object Parameters

  • @FoodSourceID
  • @Source
  • @StoreID
  • @FoodSourceTypeID
  • @AddressID

Description

Food source objects (aka donors) represent the source of food in food-in transactions.

StoreID field is used to specify store number of donors types that are grocery rescue. The StoreID field can have a null value.
